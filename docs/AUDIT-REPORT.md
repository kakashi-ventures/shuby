   - `Account#premium?` helper does not exist — needs creation
   - Child limit (free: 1) not enforced in `ChildrenController`

   - No paywall UI components
   - Stripe/payment processor not configured for Shuby plans

   - No data export (right to portability)
   - No account deletion workflow (right to erasure)
